## 📘 Definición

Las listas son una estructura de datos en Python que permite almacenar múltiples elementos dentro de una misma variable.

Los elementos se guardan entre corchetes `[]` y pueden ser modificados.

---

## 🧠 ¿Para qué sirve?

Las listas se utilizan para guardar colecciones de datos como números, textos o incluso otros objetos.

Permiten recorrer los elementos, agregarlos, eliminarlos o modificarlos fácilmente.

---

## 🧩 Sintaxis

```python
# crear una lista
frutas = ["manzana", "pera", "uva"]

# acceder a un elemento
print(frutas[0])

# agregar un elemento
frutas.append("naranja")
